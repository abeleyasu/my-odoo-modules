RingCentral Call

Click-to-dial, call control, and call logging in Odoo

Key Capabilities

  • Click-to-dial anywhere: Call directly from partners, leads, tickets, and opportunities.
  • Call control widget: Hold, mute, transfer, and manage active calls in real-time.
  • Automatic call logging: Keep a complete communication history on Odoo records.
  • Incoming call popups: Identify callers instantly with linked records.
  • RingOut support: Two-legged calling for reliable outbound calls.

Works With

Built on top of RingCentral Base and integrates seamlessly with Contacts, CRM, Helpdesk, Projects, Sales, and other Odoo apps.
