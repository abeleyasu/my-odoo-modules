Office Drive - Enterprise Document Management

A Google Drive-caliber DMS built natively for Odoo

Why Office Drive?

Office Drive brings a full-featured, enterprise-ready Document Management System directly into Odoo. Designed to match — and in many areas exceed — the usability of popular cloud drives, Office Drive keeps your data on your infrastructure while offering the collaboration, previews, and editors teams expect.

  • Native Odoo integration: Works with Odoo users, records and access controls.
  • Enterprise controls: ACLs, audit logging, password-protected share links and expirations.
  • OnlyOffice-ready: Edit Office files online with OnlyOffice integration (configurable).
  • Large file support: Chunked uploads for files up to 10 GB.
  • Wide format support: Supports 18+ file categories and 100+ file formats (images, video, audio, archives, code, Office, PDF, and more).

Key Features

Intuitive Interface

Drag-and-drop, grid/list views, rich previews, and folder navigation — familiar to end users.

Advanced Sharing

Share with users or via public links with password protection, expiration and fine-grained permissions.

Enterprise Security

ACLs, audit logs, per-file permissions and integration with Odoo's access model.

Built-in Editors

OnlyOffice support for Office formats plus Monaco editor for code files (Python, JS, XML, CSS).

Rich Previews

Preview images, PDFs, videos, audio, code and many archive formats without downloads.

Versioning & Large Files

Version history, large file chunked uploads and efficient storage with optional deduplication.
